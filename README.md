NPK 检索工具
一款基于 Python Tkinter 开发的 NPK 文件检索与管理工具，支持中文关键词映射、文件翻译标注、批量复制等功能，专为快速定位和管理 NPK 文件设计。
📋 功能特性
目录扫描：自动扫描指定目录下所有.npk 文件，记录文件名称、修改时间和完整路径
智能检索：
支持多关键词检索（空格分隔）
中文关键词映射（通过配置文件实现中文到英文关键词的匹配）
支持文件名、翻译内容双重检索
翻译管理：
为 NPK 文件添加 / 修改中文翻译标注
自动加载翻译配置文件，支持多种编码格式（GBK/GB2312/UTF-8）
文件操作：
双击文件路径直接打开文件所在位置
批量复制选中的 NPK 文件到指定目录
配置持久化：自动保存 NPK 目录路径等配置，下次启动自动加载
界面优化：
双色交替行显示，提升可读性
支持 DPI 适配，界面显示更清晰
右下角版权信息提示

⚙️ 配置说明
1. 关键词映射文件（npk_mapping.txt）
工具启动时会自动创建默认映射文件，格式如下：
plaintext
# 注释行（以#开头）
# 格式：中文关键词=英文关键词
上衣=top
上衣=top1
上衣红=topred
支持一个中文关键词对应多个英文关键词
关键词不区分大小写
修改后点击「重新加载映射表」生效
2. 翻译文件（npk.txt）
自动创建，用于存储 NPK 文件的中文翻译，格式如下：
plaintext
# 文件名 翻译（以#开头为注释）
character.npk 角色基础文件
weapon.npk 武器资源文件
支持通过「添加翻译」「修改翻译」按钮可视化编辑
自动识别文件编码（GBK/GB2312/UTF-8/UTF-8-SIG）
3. 配置文件（config.ini）
自动生成，保存程序配置：
ini
[SETTINGS]
npk_root_dir = D:\npk_files  # 上次选择的NPK根目录
📖 使用指南
基础操作
选择 NPK 目录：点击「选择 NPK 目录」按钮，选择存放 NPK 文件的根目录
文件检索：
在搜索框输入关键词（多个关键词用空格分隔）
按回车键或点击「搜索」按钮执行检索
点击「清空检索框」重置搜索条件
翻译管理：
选中列表中的文件，点击「添加翻译」输入中文标注
选中已翻译的文件，点击「修改翻译」更新标注内容
文件复制：
选中需要复制的文件（可多选）
点击「复制选中文件」，选择目标目录完成复制
其他操作：
双击列表中的文件可直接打开文件所在文件夹
修改映射文件后，点击「重新加载映射表」更新配置
检索规则
关键词不区分大小写
多个关键词为「且」关系（需同时匹配）
支持匹配：
文件名本身
翻译内容
中文关键词对应的英文映射（如输入「上衣」会匹配包含「top」的文件）
🛠️ 常见问题
Q1: 工具启动后提示 "请选择有效的 NPK 目录"
解决方案：点击「选择 NPK 目录」按钮，选择包含.npk 文件的文件夹
Q2: 中文关键词检索不到结果
检查 npk_mapping.txt 文件是否配置正确
确认关键词映射格式为「中文 = 英文」
点击「重新加载映射表」后重试
Q3: 翻译文件乱码
工具会自动尝试多种编码格式读取
若仍乱码，可手动将 npk.txt 文件转为 GBK 编码
Q4: 双击文件无法打开
确认文件路径存在且有权限访问
检查系统是否关联了.npk 文件的打开方式
